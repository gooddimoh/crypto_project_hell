Blackjack for Crypto Casino

About

Blackjack is an add-on game for the Crypto Casino application. This game can not be installed on its own without having Crypto Casino installed first.

System requirements

  • Crypto Casino 1.14.2 or higher.

Installation

Step 1

Log in to the Crypto Casino backend.

Step 2

Open Add-ons page, find the purchased add-on in the list and click Install button.

Step 3

Input the purchase code and click Proceed button.

The add-on should be automatically downloaded and installed.

Frequently Asked Questions

How to upgrade the add-on to a new version?

The application automatically checks for updates on a daily basis. If a new version is available a notice will be displayed on the Add-ons page in the backend. Simply click Update button to install the new version.

Please note that the game settings will be preserved, however any code changes will be overwritten.
