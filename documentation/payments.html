Payments for Crypto Casino

About

Payments is an add-on for the Crypto Casino application, which enables deposits and withdrawals. This add-on can not be installed on its own without having Crypto Casino installed first.

System requirements

  • Crypto Casino 1.14.2 or higher.

Installation

Step 1

Log in to the Crypto Casino backend.

Step 2

Open Add-ons page, find the purchased add-on in the list and click Install button.

Step 3

Input the purchase code (it is sent by email from 2checkout.com) and click Proceed button.

The add-on should be automatically downloaded and installed.

Frequently Asked Questions

What payment gateways are supported?

At the moment the following payment gateways are supported:

  • Paypal (only deposits)
  • Stripe (only deposits)
  • Coinpayments.net (deposits and withdrawals)
  • Ethereum via Metamask (only deposits)

How to start accepting deposits?

To accept deposits you need to configure at least one Payment gateway. First, input necessary gateway account details (such as public key, private key etc) on the Settings page, Add-on: Payments tab. Then make sure this gateway is enabled on the Payment gateways page. You also need to specify the default currency you will accept deposits in and the reference rate, which is used to convert deposited money from real currency into credits.

What are the extra steps required to enable coinpayments.net gateway?

Please sign up at coinpayments.net, obtain merchant ID, public and private keys and input them in the app settings in the backend.

After signing up with coinpayments.net you need to enable certain coins, which users are allowed to use for deposits and withdrawals. Apart from that you have to set the following permissions for the coinpayments.net API key (Account --> API keys --> Edit permissions):

  • get_basic_info
  • create_transaction
  • get_tx_info
  • balances
  • create_withdrawal
  • Allow auto_confirm = 1 in create_withdrawal (only in case you like to enable auto confirmation of withdrawals)
  • get_withdrawal_info
  • rates

For security reasons it's recommended that email confirmation is required for any withdrawals from your account (Account Settings --> Login & Security).

How to configure withdrawals?

On the Withdrawal methods page you can choose which methods are available to users when creating a withdrawal request. There are couple of methods available by default, but you are free to create your own methods. Of course, custom withdrawal methods do not support automatic processing and you will need to take care of transferring funds manually.

How are withdrawals processed?

Withdrawals can be processed manually or automatically (coinpayments.net method only).

If you want a want withdrawal request to be processed automatically you need to open the request page by clicking View button on the Withdrawals page in the backend and click Approve & Send button. An automatic transaction to transfer the requested amount of coins from your coinpayments.net account to the specified user address will be initiated (note that credits are automatically converted into coins using the reference rate and live market rates). When this transaction is completed (specific number of confirmations is received from the blockchain) coinpayments.net will send an instant payment notification (IPN) to your website and the request will be automatically marked as Completed. In order for this process to work you need to make sure that the Secret key provided in the settings is the same as in your coinpayments.net account and that external IPN requests are not blocked (e.g. by CloudFlare, proxy server or firewall).

If you want to process a withdrawal request manually you can send the requested funds directly to the client and then click Complete button to finish the withdrawal request.

To cancel a withdrawal request and return credits back to the user account click Reject.

How to upgrade the add-on to a new version?

The application automatically checks for updates on a daily basis. If a new version is available a notice will be displayed on the Add-ons page in the backend. Simply click Update button to install the new version.

Please note that the game settings will be preserved, however any code changes will be overwritten.
